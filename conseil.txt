Conseil pour les exams

- S'inscrire a l'event et au projet
- Se log avec exam exam
- Executer examshell (pas besoin de kinit)
- Toujours compiler avec les flags -Wall -Wextra -Werror

Pour vim : 

- syn on            -> permet d'afficher la syntaxe
- set nu            -> affiche le lignes
- set mouse = a     -> permet d'activer la souris
